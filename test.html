文档中的样式

前言

文档中的样式主要设计 Markdown 编辑时的一些主要样式，例如，段落、标题、图片、引用块、表格等等。这份样式不仅可以保证你可以在 Markdown 编辑器中编辑文章时，保证美观的样式输出，同时，我也写了诸多自定义样式——你可以通过在 html 标签中增加特定的类来使用它们。

块状元素

图片

Markdown 图片

所谓 Markdown 图片，也就在不使用任何容器的情况下增加图片。它操作简单，而且非常受 Markdown 用户欢迎。

Donec et elit eget nisl vulputate cursus. Pellentesque pellentesque hendrerit sapien in bibendum. Morbi mattis libero eget enim feugiat viverra.

带注释的图片

艺术书、儿童读物、科学杂志、学术期刊，以及数学书的编辑们酷爱带注释的图片。在此，我并没有使用新增的 figure 以及内嵌的 figcaption 标签，而是对 div 追加了类，以保证它能够在最老旧的浏览器上显示正常。

当你在 caption 类中为图片增加图注时，我建议把文字写在 p 标签内。这么做的好处是，你可以在图注内加入超链接、下划线、高亮，加粗等元素。

当然，虽然我同意你在图注内加入 h1、ol、blockquote，或者 table 等标签，但是，它们的显示将非常难看——因为我从一开始就不建议这么做！
