1-3 Sets and Operations on Sets. Quantifiers

Book: Mathematical Analysis
Chapter: Chapter 1: Set theory
Author: Elias Zakon

A set is a collection of objects of any specified kind. Sets are usually denoted by capitals. The objects belonging to a set are called its elements or members. We write \(x\in A\) if \(x\) is a member of \(A\), and \(x \notin A\) if it is not.

\(A=\{ a, b, c, ... \}\) means that \(A\) consists of the elements \(a, b, c, ...\) . In particular, \(A=\{a,b\}\) consists of \(a\) and \(b\); \(A = \{p\}\) consists of \(p\) alone. The empty or void set, \(\emptyset\), has no elements. Equality (\(=\)) means logical identity.

If all members of \(A\) are also in \(B\), we call \(A\) a subset of \(B\) (and \(B\) a superset of \(A\)), and write \(A \subseteq B\) or \(B \supseteq A\). It is an axiom that the sets \(A\) and \(B\) are equal (\(A=B\)) if they have the same members, i.e.,

\[ A \subseteq B \land B \supseteq A. \]

If, however, \(A \subseteq B\) but \(B \not \subseteq A\) (i.e., \(B\) has some elements not in \(A\)), we call \(A\) a proper subset of \(B\) and write \(A \subset B\) or \(B \supset A\). "\(\subseteq\)" is callded inclusion relation.

Set equality is not affected by the order in which elements appear. Thus \(\{a,b\} = \{b,a\}\). Not so for ordered pairs \((a, b)\).* For such pairs,

\[ (a,b) = (x,y) \iff a=x \land b=y, \]

but not if \(a=y\) and \(b=x\). Similarly, for ordered n-tuples.

\[ (a_1, a_2, ..., a_n)=(x_1, x_2, ..., x_n) \iff a_k = x_k, k=1,2,...,n. \]

* See Problem 6 for a definition.

We write \(\{ x \;|\; P(x) \}\) for "The set of all \(x\) satisfying the condition \(P(x)\)." Similarly, \(\{(x,y) \;|\; P(x)\}\) is the set of all ordered pairs for which \(P(x,y)\) holds; \(\{x\in A \;|\; P(x)\}\) is the set of those \(x\) in \(A\) for which \(P(x)\) is true.

For any sets \(A\) and \(B\), we define their union \(A \cup B\), intersection \(A \cap B\), difference \(A-B\), and Cartesian Product (or cross product) \(A \times B\), as follows:

  • \(A \cup B\) is the set of all members of \(A\) and \(B\) taken together:

\[ \{x \;|\; x\in A \lor x \in B\}. \]

  • \(A\cap B\) is the set of all common elements of \(A\) and \(B\):

\[ \{ x\in A \;|\; x\in B \}. \]

  • \(A-B\) consists of those \(x\in A\) that are not in \(B\):

\[ \{ x\in A \;|\; x \notin B \}. \]

Notes:

  1. Unlike the case in arithmetic, \( A - B = \emptyset \) does not implies \(B - A = \emptyset\). For if \(A \subset B\), \(A-B = \emptyset\), but \(B=A \ne \emptyset\);
  2. So, \(A-B=B-A\) iff \(A=B\), and \(A-B=B-A=\emptyset\);
  3. If \(A\) and \(B\) are disjoint, i.e., \(A \cap B = \emptyset\), \(A-B=A\) and \(B-A=B\).
  • \(A \times B\) is the set of all ordered pairs \((x,y)\), with \(x \in A\) and \(y \in B\):

\[ \{ (x,y) \;|\; x \in A, y \in B \}. \]

Similarly, \(A_1 \times A_2 \times ... \times A_n\) is the set of all ordered n-tuples \((x_1, ..., x_n)\) such that \(x_k \in A_k\), \(k=1,2,...,n\). We write \(A^n\) for \(A\times A\times ... \times A\) (\(n\) factors).

\(A\) and \(B\) are said to be disjoint iff \(A\cap B=\emptyset\) (no common elements). Otherwise, we write \(-X\) for \(S-X\), and call \(-X\) the complement of \(X\) (in \(S\)). Various other notations are likewise in use.

Examples.

Let \(A=\{ 1,2,3 \}\), \(B=\{2,4\}\). Then

\[ \begin{aligned} A \cup B &= \{1,2,3,4\},\\ A \cap B &= \{2\},\\ A-B &=\{1,3\},\\ A\times B &= \{ (1,2),(1,4),(2,2),(2,4),(3,2),(3,4) \}. \end{aligned} \]

If \(N\) is the set of all naturals (positive integures), we could also write

\[ A=\{x\in N \;|\; x<4 \} \]

Theorem 1.

\[ \begin{aligned} &\text{(a) } A \cup A = A,\; A \cap A = A; \\ &\text{(b) } A \cup A = B \cup A,\; A \cap B = B \cap B; \\ &\text{(c) } (A \cup B)\cup C = A \cup (B \cup C), (A \cap B) \cap C = A \cap (B \cap C); \\ &\text{(d) } (A \cup B)\cap C = (A \cap C) \cup (B\cap C); \\ &\text{(e) } (A \cap B)\cap C = (A \cup C) \cap (B\cup C).\\ \end{aligned} \]

The proof of (d) is sketched in Problem 1. The rest is left to the reader.

Because of (c), we may omit brackets in \(A \cup B \cup C\) and \(A \cap B \cap C\); similarly for four or more sets. More generally, we may consider whole families of sets, i.e., collections of many (possibly infinitely many) sets. If \(\cal M\) is such a family, we define its union, \(\bigcup \cal M\), to be the set of all elements \(x\), each belonging to at least one set of the family. The intersection of \(\cal M\), denoted \(\bigcap \cal M\), consists of those \(x\) that belong to all sets of the family simultaneously. Instead, we also write

\[ \bigcup \{ X \;|\; X \in \cal M \} \]

and

\[ \bigcap \{ X \;|\; X \in \cal M \}, \]

respectively.
